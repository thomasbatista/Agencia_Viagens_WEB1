# ✈️ Agência de Viagens — Projeto WEB1 (IFSP)

Site demonstrativo criado como trabalho da disciplina **Desenvolvimento Web 1 (WEB1)** do IFSP.

Funcionalidades

- Log de Sistema: registra no console quando o script é carregado.
- Carrossel de destinos: controla quais slides aparecem e permite navegação manual.
- Validação de formulário: evita envio com campos vazios e limpa o formulário após envio simulado.
- Toggle de detalhes de pacotes: mostra/esconde seções de benefícios adicionais dentro de `pacotes.html`.
- Saudação no rodapé: modifica o conteúdo de `#footer-msg` conforme a hora atual.

Mapa rápido de arquivos

- `index.html` — página inicial com hero e destaques.
- `destinos.html` — carrossel e tabelas de hotéis/roteiros.
- `pacotes.html` — listagem detalhada de pacotes com botões de detalhes.
- `sobre.html` — informações institucionais.
- `contato.html` — formulário de contato.
- `style.css` — todos os estilos e media queries.
- `script.js` — lógica do carrossel, validação de formulário e toggle do menu.
